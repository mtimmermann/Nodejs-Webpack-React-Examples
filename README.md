# Static Website Boilerplate

Static website boilterplate project template built using Webpack.

##### Server
* Node.js
* Express 4

##### Client JS
* Webpack
	* ESLint
* React 15.6
* ES6

#### Client Styles
* Bootstrap
* Font Awesome
* Sass

----

Start development
```sh
npm run start
```

Build the dist folder
```sh
npm run
```
