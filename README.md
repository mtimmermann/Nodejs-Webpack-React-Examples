# Static Website Boilerplate

Static website boilterplate project template built using Webpack.

* Webpack
	* ESLint
 	* Browsersync live client source to browser reloading
* React
* ES6
* Bootstrap
* Font Awesome
* Sass

----

Start development
```sh
npm run start
```

Build the dist folder
```sh
npm run
```
